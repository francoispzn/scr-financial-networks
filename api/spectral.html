scr_financial.network.spectral.find_spectral_gap(eigenvalues, min_index=1, max_index=None, adjacency_matrix=None)[source]¶

Identify the spectral gap in the eigenvalue spectrum using the Erdos-Renyi null model test from Schmidt, Caccioli & Aste (2025).

For each gap index i, we compare the observed gap against a distribution of gaps from an ensemble of Erdos-Renyi random graphs with the same number of vertices and expected average degree. A gap is flagged as a candidate if its empirical one-tailed p-value is below alpha=0.01.

If no statistically significant gap is found, falls back to the largest gap in [min_index, max_index].

Parameters:

  • eigenvalues (ndarray) – Array of eigenvalues (sorted ascending).

  • min_index (int) – Minimum gap index to consider. Defaults to 1.

  • max_index (int | None) – Maximum gap index to consider. Defaults to None (half of spectrum).

  • adjacency_matrix (ndarray | None) – Optional adjacency matrix for computing exact average degree for the ER null model. If None, the degree is estimated from the eigenvalue spectrum.

Returns:

Tuple of (gap index, gap size).

Return type:

Tuple[int, float]

References

Schmidt, Caccioli & Aste, “Spectral coarse graining and rescaling for preserving structural and dynamical properties in graphs”, Phys. Rev. E 112, 034303 (2025).
